## üé≠ Demo Accounts

### Student Account
**How to login:**
1. Click "Login" in the navigation bar
2. Select "Student" role
3. Enter any email (e.g., student@test.com)
4. Enter any password
5. Click "Login"

**What you'll see:**
- Student Dashboard with applications, fees, exams, offers
- Sample data showing 3 applications, 2 fee statuses, 3 exam scores
- Offer letters from Stanford and Toronto
- Academic reports and full profile

### Admin Account
**How to login:**
1. Click "Login"
2. Select "Admin" role
3. Enter: admin@globaledu.com
4. Enter any password
5. Click "Login"

**What you'll see:**
- Admin dashboard with platform statistics
- Sections for managing applications, students, universities
- Chat message management

### University Account
**How to login:**
1. Click "Login"
2. Select "University" role
3. Enter: admissions@university.edu
4. Enter any password
5. Click "Login"

**What you'll see:**
- University portal with application statistics
- Sections for reviewing applications and managing programs

## üìã Key Features to Test

### 1. Browse Universities (No Login Required)
- **Go to:** Home page (default)
- **Try:** 
  - Filter by country (USA, UK, Canada, etc.)
  - Filter by program level
  - Search for specific universities
  - Click university cards to see details

### 2. Submit Application (No Login Required, but recommended)
- **Go to:** Click "Apply Now" button
- **Try:**
  - Fill out the comprehensive application form
  - All sections: Personal Info, Education, Test Scores, Program Selection
  - Submit and see success notification

### 3. Student Dashboard (Login Required)
- **Go to:** Login as student, then click "Dashboard"
- **Try:**
  - Overview: See statistics and recent activity
  - Applications: View 3 sample applications with different statuses
  - Fees: Check payment status with progress bars
  - Exams: View test scores (TOEFL, GRE)
  - Offers: See offer letters from universities
  - Academic: Check academic records
  - Profile: View complete student profile

### 4. Live Chat (Available on All Pages)
- **Find:** Floating chat button (bottom right corner)
- **Try:**
  - Click to open chat window
  - Type a message and press Enter or click Send
  - Get automated bot response
  - Messages are saved in browser storage

### 5. Admission Process (No Login Required)
- **Go to:** Click "Admission Process" in navigation
- **Try:**
  - Read through 8-step admission process
  - Each step has detailed checklist
  - Click "Start Your Application" at the bottom

### 6. Filters & Search (No Login Required)
- **Go to:** Home page
- **Try:**
  - Country filter: Select USA, UK, Canada, Australia, or Germany
  - Program filter: Select Undergraduate, Postgraduate, or PhD
  - Search box: Type university name or location
  - Filters work together in real-time

## üêõ Troubleshooting

**Issue: Page not loading properly**
- Solution: Clear browser cache and reload
- Solution: Check browser console for errors
- Solution: Try different browser

**Issue: Login not working**
- Solution: Use any email and password (it's a demo)
- Solution: Make sure to select a role
- Solution: Check localStorage is enabled

**Issue: Chat not responding**
- Solution: Refresh the page
- Solution: Messages save in localStorage
- Solution: Clear browser data to reset

**Issue: Filters not working**
- Solution: Reload the page
- Solution: Clear search box and try again
- Solution: Check browser console for errors
